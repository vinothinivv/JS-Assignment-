Difference between var, let and const

Variable

var means Variable, which is globally scoped. Globally Scoped means, var is declared outside the block of function.

Above, we can give the same the name is declared for two variable. But, in the output, 'hello vinothini' is displayed. Beacuse, var is globally scoped.

Let

Anything within curly braces is a block. let is live in the block

Using hello1 outside its block (the curly braces where it was defined) returns an error. This is because let variables are block scoped .

Constant

Variables declared with the const maintain constant values.Every const declaration, therefore, must be initialized at the time of declaration.
