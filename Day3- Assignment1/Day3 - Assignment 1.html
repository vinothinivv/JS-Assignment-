Types of Console Funciton

In Javascript, console is mainly to used to debugging in the browser. Different type of console are :

  • console.log()
  • console.error()
  • console.warn()
  • console.table()
  • console.count()
  • console.time() and console.timeEnd()

Console.log()

It is mainly used to print the output to the console. We can put any type inside the log(), be it a string, array, object, boolean etc.

Console.error()

Used to log error message to the console. Useful in testing of code. By default, the error message will be highlighted with red color.

Console.warn()

Used to log warning message to the console. By default, the warning message will be highlighted with yellow color.

Console.table()

This method allows us to generate a table inside a console. The input must be an array or an object which will be shown as a table.

Console.count()

This method is used to count the number that the function hit by this counting method.

console.time() and console.timeEnd()

By the use of the time() and timeEnd() methods, amount of time spend by a block or a function can be calculated.
